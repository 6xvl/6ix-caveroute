# caveman-why

**Expand the last caveman answer back to full prose.**

caveman speak save token. Sometime terse answer too short — meaning unclear, or
you want the reasoning it dropped. `/caveman-why` bring back full version.

## Use it

Say any of:

- `/caveman-why`
- "caveman-why"
- "explain that fully" / "expand that" / "decompress that"
- "what did that mean"

Claude rewrites its **most recent** answer in normal, complete prose — restoring
articles, sentences, and the *why* — while keeping every technical fact, value,
and code block exactly the same.

## What it does NOT do

- It does **not** turn caveman off. The next answer is terse again. To switch
  off for good, say "stop caveman" or "normal mode".
- It does **not** add new information or re-run anything — it only decompresses
  an answer already given. If that answer was wrong, Claude says so; it does not
  hide a correction inside the expansion.

## Why

One rock too small, sometime. `/caveman-why` give big rock back — just that one
time. Every other turn stay cheap.
